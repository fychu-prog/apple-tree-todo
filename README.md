# 🍎 蘋果待辦樹 (Apple Todo Tree)

一個基於 Matter.js 物理引擎與 PWA 技術開發的互動式待辦清單 (Todo List)。透過視覺化的蘋果成長與採收過程，讓您的生產力管理變得充滿樂趣與重量感。

![Apple Todo Tree Demo](icon.svg)

## ✨ 核心特色

- **🍎 物理採收系統**：點擊樹上的蘋果，它會隨著重力掉入下方的金屬籃子中。
- **📱 行動優先設計**：完美適配手機屏幕，支援觸控拖拽與單手輸入。
- **📳 陀螺儀互動**：支援手機重力感應，籃子裡的蘋果會隨著您傾斜手機而滾動。
- **🧺 收穫布告欄**：精緻的公告欄設計，記錄您一整天的豐功偉業。
- **🚀 PWA 支援**：可安裝至手機桌面，具備原生 App 級別的啟動體驗。
- **🌳 動態負載機制**：當樹上累積超過 22 個任務時，系統會溫馨提示您先採收工作果實，避免過勞。

## 🛠️ 技術堆疊

- **Engine**: Matter.js (2D 物理模擬)
- **Styling**: Vanilla CSS (現代佈局、玻璃擬態)
- **Logic**: Vanilla JavaScript (ES6+)
- **Storage**: HTML5 LocalStorage
- **Service Worker**: PWA 離線緩存

## 🚀 快速開始

1. 克隆此儲存庫。
2. 使用任何靜態網頁伺服器開啟 `index.html` (例如 `npx http-server`)。
3. 在手機瀏覽器中開啟並選擇「新增至主畫面」。
